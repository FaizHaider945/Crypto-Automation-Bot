# 🕷️ Spidey Bot: Crypto Market Analyzer
An automated Python bot that tracks Bitcoin market sentiment and executes virtual trades for performance tracking.

## 🚀 Key Features
- **Data Sources:** Real-time data from @Bitget (Funding) & @Binance (OI).
- **Intelligence:** Signal generation powered by @GoogleGemini logic.
- **Session Reports:** Automatically tracks Win Rate and Net PNL.
- **Instant Alerts:** Live updates delivered to @Discord via Webhooks.

## 🛠️ Tech Stack
- **Language:** Python 🐍
- **Hosting:** PythonAnywhere ☁️
